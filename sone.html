Chain up all four APIs: click the buttom 'Click To Generate Random Anime' to generate an anime using Anime API, which will display the Name, Image and Start Date of the anime. Based on the Start Date of the anime, it will generate the holiday of that day (if there isn't any holiday on that day, it will shows 'No holiday found for this date,' then you can click the buttom again to regenerate another anime until it matches up with a Holiday) using Holiday API. Used the name of the holiday as keywords, it will search through News API to find the first news that matches the name of the holiday. In the News API section, it will displays the Title, Published Date, Header Image, and a short description of that news; meanwhile, by clicking the image of the news, you will be direct to the actual News post. Furthermore, it will use the Published Date of that News to apply into NASA API and get the astronomy image of the news's published date; in the meantime, by clicking the NASA astronomy image, you will can view the text that correlated to NASA astronomy image.
